»a« logic

  • A language
  • A set of axioms
  • Axioms are trivially theorems
  • Theorems: words of the language made from combining theorems

Classical logic

  • Language: binary and (\(\wedge\)), binary or (\(\vee\)), boolean values
  • Axioms, e.g. \(A \wedge B \Rightarrow A\)
  • Axioms are trivially theorems
  • Theorem, e.g. \(A \wedge B \Rightarrow B \wedge A\)

Some classical axioms

$$\frac{A \quad B}{A \wedge B} \quad (\wedge I) \qquad \frac{A \wedge B}{B} \quad (\wedge E_R) \qquad \frac{B}{A \vee B} \quad (\vee I_R)\qquad$$

Falsehood

$$ \frac{A \quad \neg A}{\bot}~(\bot I) \qquad \frac\bot B~(\bot E) $$

Truth

$$ \frac \cdot\top~(\top I) $$

LEM

$$\frac\cdot{A \vee \neg A}$$

LEM-based theorems

  • p is prime or not: \(\forall p.~\text{prime}(p) \vee \neg \text{prime}(p)\)
  • Riemann hypothesis: \(\text{holds} \vee \neg \text{holds}\)
  • Continuum hypothesis: \(\text{holds} \vee \neg \text{holds}\)
  • Thor: \(\text{exists} \vee \neg \text{exists}\)

Classical logic: truth is discovered, not invented

Constructive logic

No LEM!
  • p is prime or not: \(\forall p.~\text{prime}(p) \vee \neg \text{prime}(p)\)
  • Riemann hypothesis: \(\text{holds} \vee \neg \text{holds}\) – NOPE
  • Continuum hypothesis: \(\text{holds} \vee \neg \text{holds}\) – NOPE
  • Thor: \(\text{exists} \vee \neg \text{exists}\) – NOPE

Constructive logic: truth is invented, not discovered

Application: type systems, e.g. Java, Haskell

Substructural logic

  • Linear: Each proposition must be used exactly once
  • Affine: Each proposition must be used at most once
  • Relevant: Each proposition must be used at least once

Application: resource management, e.g. Rust

Temporal logic

Propositions are only true for some period

  • I am 29 years old
  • This algorithm never deadlocks

Application: database design
